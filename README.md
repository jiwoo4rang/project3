# 🤖 AI 지방의회 운영 가이드 시스템

지방의회 운영과 관련된 정보를
AI 기반으로 질문하고 답변받을 수 있는 웹 챗봇 시스템입니다.

---

## 🚀 프로젝트 소개

이 프로젝트는 지방의회 운영 가이드 및 의정 정보를
사용자가 쉽게 검색하고 이해할 수 있도록 만든 AI 챗봇 서비스입니다.

사용자는 자연어로 질문을 입력하면,
AI가 관련 정보를 기반으로 답변을 제공합니다.

---

## 🛠️ 사용 기술

* HTML / CSS / JavaScript
* REST API (AI 서버 연동)
* LocalStorage (테마 저장)
* Fetch API (비동기 통신)

---

project/

## 🎯 주요 기능

### 💬 AI 챗봇 기능

* 사용자 질문 입력 → AI 응답 출력
* 대화 히스토리 유지
* 서버 기반 응답 처리

### 🎨 UI/UX 기능

* 다양한 테마 변경 기능 (기본 / 블루 / 레드 / 그린 / 다크)
* 채팅 인터페이스 (말풍선 UI)
* 타이핑 애니메이션 효과
* 모바일 반응형 UI

### ⚙️ 기타 기능

* 사이드바 열기 / 닫기
* 채팅 초기화 기능
* 텍스트 확대 기능
* 실시간 서버 상태 표시
